# 🍋 Little Lemon Restaurant Website

This is a front‑end project I built using **HTML** and **CSS**.  
It’s a multi‑page restaurant website with a home page, menu, reservation form, and about section.  
I created it as part of my learning journey in web development.

## 🌟 Features
- Responsive layout across all pages
- Menu page with dish highlights
- Reservation page with a simple booking form
- About page with text and image layout
- Clean navigation and footer design
- Responsive design across all pages (works smoothly on desktop and mobile)
- **Menu page** showcasing dishes
- **Reservation page** with a booking form
- **About page** with restaurant details
- Smooth **hover effects** on navigation and cards
- Basic **CSS animations** for interactive elements
